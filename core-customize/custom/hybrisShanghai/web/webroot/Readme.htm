Readme

This simple tool is aiming to help beginners of hybris Platform to have a better understanding of the type system used within hybris, and it might be helpful in constructing complex FlexibelSearch or even ImpEx queries. It is recommended to use it together with HMC and Admincockpit to get familiar with the types.

 

All the types are listed on the left panel, it is loaded dynamically from the platform by using FlexibleSearch and once loaded the number in the title bar indicate the total number of the types found. You can sort the types in different direction and hide certain columns by clicking the table head and search certain types by using the search toolbar.

 

 

 

 

Click on the types on right side, the attributes of it will be displayed on right panel which could be collapsed at any time.

 

 

 

 

Right click on the types a pop-up menu will be shown, you can:

1.       Show the attributes of the type (same effect as clicking on the type).

2.       Search the supertype of current selection in the grid in order to find out its details.

3.       Search all persisted items of the selected type excluding/including the items of its subtypes.

 

 

All items are listed in a new items- window, which support live search and paging for better response time.

You can view the values of each attributes of an item by a simple click on it

 

 

By clicking a Show Hierarchy button on the left panel you will have a tree diagram of the entire type system. You can 1) expand and collapse the tree by using the tool buttons located on the right corner of the window. 2) Zoom in/out the scree by scrolling your mouse wheel. 3) Drag to move the diagram. 4) if the tree is not fully expanded, the nodes having child nodes will be displayed in blue color, you can click each node to expand it.
